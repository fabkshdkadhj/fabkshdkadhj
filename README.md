# Business Tool Lab

I build small, practical workflow tools for people who need less manual work and cleaner daily operations.

Current focus:

- Spreadsheet cleanup and report preparation
- Lightweight business workflow automation
- CSV/order/inventory reconciliation samples
- Small internal tools for repetitive office tasks

I prefer boring useful tools over shiny demos: clear inputs, clear outputs, and something a real person can use the same day.

## What I am building

- [`business-tool-lab`](https://github.com/fabkshdkadhj/business-tool-lab): small business reporting, data cleanup, reconciliation, inventory, and follow-up workflow samples.
- Public demo: https://fabkshdkadhj.github.io/business-tool-lab/
